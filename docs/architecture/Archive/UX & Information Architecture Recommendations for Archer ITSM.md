# UX & Information Architecture Recommendations for Archer ITSM

# UX Recommendations

## User Research Findings

**Most Hated:**

* Too many clicks (ServiceNow)
* Slow performance
* Outdated design
* Fragmented settings (Jira)
* Poor mobile

**Most Loved:**

* Modern clean UI (Freshservice G2: 4.6/5)
* Grafana visualization
* Unified search
* Guided workflows

## Archer UX Principles

1. SPEED - Target under 100ms, React+Vite
2. SIMPLICITY - Fewer clicks, opinionated defaults
3. MODERN - Purple Glass, Fluent UI 2
4. KEYBOARD-FIRST - Cmd+K palette
5. MOBILE-EQUAL - Responsive, full features
6. CONTEXTUAL - Ticket to asset to metrics seamlessly

## Navigation

* Unified sidebar
* Global search (Cmd+K)
* Breadcrumbs
* Recent items

## Key Flows

* Ticket creation with AI suggestions
* Incident response with auto-RCA
* Asset discovery with topology
