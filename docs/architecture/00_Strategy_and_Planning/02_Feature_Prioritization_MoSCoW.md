# MoSCoW Feature Prioritization

## Must-Haves (MVP - Phase 1)

### ITSM Core

* Multi-channel ticketing (email, portal, chat)
* SLA management with escalations
* Workflow automation (no-code)
* Knowledge base
* Service catalog
* Mobile apps (responsive web)
* RBAC
* Reporting

### CMDB

* Auto-discovery (API-based for cloud)
* Asset inventory
* Relationship mapping
* Dependency visualization
* Change impact analysis

### Monitoring (Basic)

* Metrics collection
* Alerting
* Dashboards
* Alert to Ticket automation

### UX

* Modern fast UI (Purple Glass)
* Global search
* Keyboard shortcuts
* Dark mode

## Should-Haves (Phase 2)

### ITSM Advanced

* Change, problem, release management
* Project management

### AI (Phase 2A)

* Intelligent triage
* Similar ticket detection
* KB suggestions
* Anomaly detection

### CMDB Advanced

* Agentless discovery (SSH, WMI)
* Application dependency mapping
* Service mapping

### Monitoring Advanced

* Distributed tracing
* Log management
* Automated RCA
* Forecasting

## Could-Haves (Phase 3)

### AI Advanced

* Conversational ticket creation
* Auto-remediation
* Predictive alerting
* Multi-language (40+)

### ESM

* HR service desk
* Facilities management

## Won't-Haves

* Full HR platform (compete with Workday)
* CRM (compete with Salesforce)
* ERP (compete with SAP)
